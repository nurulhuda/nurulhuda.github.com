Menggambar Spirograf di Indesign CS3

23 Maret 2013 | Comments

Sebaik apapun sebuah aplikasi dibuat, sudah pasti tidak akan bisa memenuhi semua kebutuhan penggunanya yang berbeda-beda latar belakang dan kebutuhannya. Untuk itulah aplikasi-aplikasi besar semacam Adobe Indesign menyediakan sarana berupa script engine. Dengan adanya sarana tersebut pengguna bisa membuat program kecil sesuai dengan kebutuhannya. Program kecil inilah yg disebut dengan script.

Script akan sangat berguna ketika 1) kita harus melakukan suatu pekerjaan yang berulang-ulang dan 2) ketika kita tidak mungkin melakukan pekerjaan tsb secara manual. Sebagai contoh, untuk membuat gambar spirograf seperti gambar di atas bisa dibilang tidak mungkin jika dilakukan secara manual.

Kali ini aku mempersembahkan sebuah script untuk menggambar spirograf di Indesign CS3. Sebetulnya script ini adalah karya saudara saya, Peter Kahrel. Hehehe… Aku hanya membuat interface-nya saja supaya lebih mudah digunakan. Kamu bisa mengunduh script tsb di sini. Caranya: klik kanan pada link tsb lalu pilih “Save link as…”. Jangan lupa komentarnya ya….
