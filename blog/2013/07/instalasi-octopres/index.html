Instalasi Octopres di Debian Wheezy

24 July 2013 | Comments

Octopress adalah sebuah kerangka kerja (framework) yang dirancang untuk Jekyll, yaitu mesin blog yang mengubah file teks menjadi halaman2 blog statis. Untuk mulai ngeblog dengan Jekyll kita harus bekerja extra keras karena harus membuat sendiri template, css, javascript, dan konfigurasi lainnya. Namun dengan Octopress semuanya jadi mudah.

Instalasi Octopress

Sebelumnya, pastikan paket2 berikut ini telah terpasang dengan baik.

  • git
  • build-essential
  • libssl-dev
  • zlib1g-dev

Unduh kode sumber Octopress dengan git

Perintah di atas akan mengunduh kode sumber Octopress ke dalam folder octopress. Sedangkan perintah pada baris 3 untuk mengetahui versi ruby yang digunakan. Versi ruby ini akan diperlukan pada saat instalasi ruby.

Selanjutnya adalah instalasi ruby menggunakan rbenv.

Instal rbenv

Instal ruby

1.9.3-p0 adalah versi ruby yang akan diinstal. Sesuaikan versi ruby ini dengan isi file octopress/.rbenv-version. Butuh waktu agak lama untuk proses download dan kompilasi. Tunggu sampai proses kompilasi dan instalasi selesai.
